# Protocol references

This project is an independent implementation of publicly documented behavior. No firmware source code is included.

The NanoVNA-H/H4 SD-card shell behavior was verified against:

- DiSlord/NanoVNA-D: https://github.com/DiSlord/NanoVNA-D
- hugen79/NanoVNA-H: https://github.com/hugen79/NanoVNA-H

The general USB screen-capture behavior was cross-checked against:

- Ho-Ro/nanovna-tools: https://github.com/Ho-Ro/nanovna-tools
- NanoVNA-Saver/nanovna-saver: https://github.com/NanoVNA-Saver/nanovna-saver

NanoVNA and related product names belong to their respective owners. This project is not an official NanoVNA application.
